Hugo is a static site generator. It is very flexible and members of the community contribute high quality themes so that those of us without a creative bone in our body can generate websites with high visual quality Why a static site generator? A static site generator has the benefits of a view templating system, but the cost of assembly for the template is completed before the site is deployed. This allows for effective static hosting and no cost caching.

project

Each client connects to the server and communicates over a UDP socket. Each client streams commands to the server, and the server broadcasts the current State of the Game to all of the connected clients.
